# Bank Customer Balance Analysis System

A C++ program that analyzes bank account balances for 5 customers.

## Features
- Classifies accounts as Sufficient, Low, or Very Low balance
- Calculates total and average balance across all customers
- Finds highest and lowest balance
- Counts customers in each balance category

## How to Run
1. Compile: g++ Bank_Customer_Balance_Analysis_System.cpp -o bank
2. Run: ./bank

## Tech Used
C++ — arrays, loops, conditionals
